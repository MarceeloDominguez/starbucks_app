## Run Locally

Clone the project

```bash
  git clone https://github.com/MarceeloDominguez/starbucks_app.git
```

Go to the project directory

```bash
  cd starbucks_app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
